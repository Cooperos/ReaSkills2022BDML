# Описание

### Модуль 4. Разработка математического аппарата.

Подробное описание задания:
1.	Выбрать и обосновать 3 алгоритма регрессии.
2.	Обучить выбранные методы и вывести полученные метрики.
3.	Выбрать и обосновать один из алгоритмов машинного обучения.
4.	Выполнить подбор гиперпараметров модели с помощью метода подбора.
5.	Скопировать и доработать пайплайн из прошлого модуля.
6.	Интегрировать получившаяся модель в пайплайн с прошлого модуля.
7.	Выполнить предсказание testcsv.
8.	Файл предсказаний сохранен на диске.

Выходные данные по модулю:
1.	Отчет в формате HTML/PDF.
2.	Файл .ipynb, полностью исполняемый и без ошибок.
3.	Файл предсказаний.

# Description

### Module 4. Development of mathematical apparatus.

Detailed description of the task:
1. Select and justify 3 regression algorithms.
2. Train the selected methods and display the resulting metrics.
3. Select and justify one of the machine learning algorithms.
4. Select the hyperparameters of the model using the selection method.
5. Copy and modify the pipeline from the previous module.
6. Integrate the resulting model into the pipeline from the previous module.
7. Run testcsv prediction.
8. The prediction file is saved on disk.

Module output:
1. Report in HTML/PDF format.
2. .ipynb file, fully executable and without errors.
3. Prediction file.
